# 🇷🇺 RU

Административный Web API на ASP.NET Core для управления пользователями, аутентификации и авторизации.  
Проект предоставляет RESTful эндпоинты для регистрации, входа, получения JWT-токена, а также админские функции: просмотр списка пользователей, изменение ролей, блокировка/разблокировка
Также разработан Frontend на Next.js для наглядности

---

## Стек технологий

- .NET 10 (ASP.NET Core Web API)
- Entity Framework Core ORM + PostgreSQL
- JWT
- Serilog (логирование в файлы и консоль)
- Rate Limiting
- Docker + Docker Compose

---

### Требования
- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- [PostgreSQL](https://www.postgresql.org/)
- [Docker](https://www.docker.com/)

# 🇺🇸 EN

Administrative Web API built with ASP.NET Core for user management, authentication, and authorization.  
The project provides RESTful endpoints for registration, login, JWT token issuance, and administrative functions: viewing user list, changing roles, blocking/unblocking users.  
A frontend example built with Next.js is also included for demonstration purposes.

---

## Technology Stack

- .NET 10 (ASP.NET Core Web API)
- Entity Framework Core ORM + PostgreSQL
- JWT
- Serilog (logging to files and console)
- Rate Limiting
- Docker + Docker Compose

---

### Prerequisites
- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- [PostgreSQL](https://www.postgresql.org/)
- [Docker](https://www.docker.com/)
